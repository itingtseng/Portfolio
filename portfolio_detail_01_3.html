iPlants

Responsive

Description

iPlants is a application platform database management system that provide service for agribusiness and consumer by using the cultivation recorded by farmer. We using government open data to notify farmer them and cloud computing system for agribusiness to manage recording data. We provide transparent and visualization data for the consumer to educate them the knowledge from farm to table.

awesome projects
my work

  • Concept Map

    In the initial of the product development, we write down every data we can interface and goal for each roles. Try to find out our features and market segmentation. By drawing affinity diagram, we understand the relationship between every data and then we can decide which data should we need.

  • Process Flow Diagram

    After we figure out the whole database then we can start to think about the process of data and how can we manage or apply them. We can realize how these data impact users and connect them to the needs of users. Then we can make a contrast with the system which is we trying to improve.

  • Ecosytem Diagram

    In this process, we visualize every role in our ecosystem. Because the whole product impact not only farm, agribusiness, and consumer, as well as the certification body and package factory. Illustrating the conceptual relationship between various system touchpoints, therefore, the product is gradually formed.

  • Storyboarding

    After the analysis of data, it is time to design. We disassemble each user in our ecosystem and try to research the contextual of using. By implementing out research, we using the storyboarding to make our user want to use. This is one of the example that I try to use game interface to attract our farmer to record cultivation.

Inforgraphic

This infographic was from my sketch to be finished by visual designer. I try to convey our goal and roles in iPlants to designer. We have discussed over times and consult other similar style.

  • Easy to use

    For farmer, there is no more form to write down. And say goodbye to other hard-to-use recording app.

  • Easy to manage

    For agribusiness, all the recording data are on the cloud. All they have to do is print out.

  • Easy to understand

    For consumer, we try to use visualization diagram to tell them the whole story. Besides, all is transparent.
